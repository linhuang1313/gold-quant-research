| 《一个策略的自白》 | 策略开发全流程：想法→回测→过拟合评估→模拟→实盘 | **已实践** — R2-R86 完整流程 |

**关键公式 — PBO (CSCV)**:
```
1. 构建 T×N 矩阵 M (T个时间期 × N组参数的日收益率)
2. 将 T 行分为 S 块
3. 对 C(S, S/2) 种 IS/OOS 划分:
   a. IS 中选 Sharpe 最高的参数 n*
   b. 看 n* 在 OOS 中的排名 rank_oos
   c. 计算 λ = rank_oos / N
4. PBO = P(λ < 0.5) — 即 IS 最优参数在 OOS 中低于中位数的比例
```

**核心见解 — 我们的双 PBO 设计**:
- **PBO-Perturb**: 用 ±20% 随机扰动变体，回答"参数空间是平原还是尖峰"
- **PBO-CSCV**: 用网格搜索参数，回答"选择偏差有多严重"
- 两者互补，不是替代关系。我们的扰动方法更像稳健性检验，网格方法更接近原始论文

### 1.2 金融数学基础

| 文章 | 核心内容 | 与我们项目的关系 |
|------|---------|-----------------|
| 《金融时间序列分析：基础篇》(p/38320827) | 平稳性、自相关、ARMA、GARCH、波动率聚集。非平稳序列需差分/取对数收益率转化 | ATR 是 GARCH 思想的简化实践；Vol-sizing `lot = risk/(ATR×PV)` 是波动率时变的应用 |

| 《布朗运动、伊藤引理、BS公式（后篇）》(p/38294971) | BS 公式推导、风险中性定价、Delta 对冲、Greeks | 理论基础：BS 假设 σ 恒定，但现实 σ 时变 → GARCH / ATR |
| 《凯利公式，从赌场到量化投资》(p/38279377) | Kelly Criterion: f* = (bp-q)/b，最优下注比例使对数增长率最大化。实践中使用半凯利(Half-Kelly)降低方差 | **核心应用**: 半凯利理念已融入我们的仓位管理思考。Ed Thorp: "半凯利损失约 25% 增长率，方差减半" |

**凯利公式关键推论**:
```
f* = (bp - q) / b

其中: f*=最优下注比例, b=盈亏比, p=胜率, q=败率(1-p)

实践要点:
- 使用 Half-Kelly (f*/2) 应对参数估计误差和肥尾分布
- 凯利公式对参数极其敏感: 胜率偏移 5% → 最优仓位差一倍
- 永远不要全仓 — 全仓策略的破产概率趋近 100%
```

### 2.1 量化交易系列文章（共 7 篇 + 衍生内容）

#### 系列（一）：你赚的每一分钱，到底从哪来？

**核心概念**:
- **Alpha 来源**: 信息优势、分析优势、结构优势、行为优势
- **市场微观结构**: 做市商、订单簿、流动性
- **期望值思维**: 策略的本质是正期望值的重复博弈
- **灵魂拷问**: 你的 Alpha 来源是什么？赚的是谁的钱？为什么现在还能赚到？

**对我们的启示**: 我们的 Keltner Channel 策略 Alpha 来源是行为优势（黄金市场散户比例高，趋势持续性强）和结构优势（低频交易不需要速度竞赛）

#### 系列（二）：三大经典策略，为什么有人靠它赚了 200 年？

**动量策略 (Momentum)**:
- 核心假设: 过去涨得好的继续涨，跌得惨的继续跌
- 学术证据: Jegadeesh & Titman (1993) 年化超额 ~12%；Moskowitz et al. (2012) 证明 200+ 年有效
- 有效原因: 反应不足(Underreaction)、确认偏差、羊群效应、处置效应
- **致命弱点**: 动量崩溃(Momentum Crash) — 2009年3月一个月亏 40%+
- **甜蜜区**: 1-12 个月回望期；1天内和3年以上反而是均值回归

**均值回归 (Mean Reversion)**:
- 核心假设: 价格偏离"正常水平"后有回归倾向
- **有效前提**: ①标的有稳定均值 ②偏离驱动力是暂时的 ③标的有足够流动性和存活概率
- **最大风险**: 你以为在抄底，实际在接飞刀 — "均值本身可能在移动"
- 经典应用: 配对交易(Pairs Trading)
- **2007 量化地震**: 策略过于拥挤，一家清仓引发多米诺效应

**统计套利 (Statistical Arbitrage)**:
- 核心思想: 大量分散的小额正期望值押注 + 大数定律 → 稳定收益
- "量化基金把自己变成赌场"
- 典型流程: 因子分解 → 组合构建(市场/行业/因子中性) → 执行风控
- 黄金时代(90s-2000s)已过，Alpha 衰减，参与者爆炸增长

**统一视角**:
```
超短期 (日内-数日)  → 微观结构效应（流动性、做市商行为）
短期   (1-12 个月)  → 动量效应（行为偏差导致的反应不足）
中期   (1-3 年)     → 过渡/混合区域
长期   (3-5 年+)    → 均值回归（过度反应的修正）
```

**对我们的启示**:
- 我们的 PSAR/TSMOM/KC 本质上是**动量策略**在黄金上的实现
- R21-S4 已验证: 极端反转(均值回归) 81 种组合全负 Sharpe → 黄金趋势性太强，不适合逆向
- 动量和均值回归不矛盾，是不同时间尺度的同一硬币两面

#### 系列（三）：90% 的人亏钱，不是因为策略差，是因为不会管风险

**亏损的不对称性**:
```
亏 10% → 回本需 +11.1%
亏 30% → 回本需 +42.9%
亏 50% → 回本需 +100%
亏 90% → 回本需 +900%
```

**凯利公式实践**:
- 数学最优下注比例: `f* = (bp-q)/b`
- **半凯利(Half-Kelly)**: 实践中标准做法，损失 ~25% 增长率但方差减半
- Ed Thorp: "Kelly betting tells you the best you can do. But half-Kelly costs you only about 25% of the growth rate while cutting the variance in half."

**三层防御体系**:
```
第一层 (仓位管理 - 日常运营):
  - 半凯利或更保守的下注比例
  - 单个头寸不超过总资金 2%-5%
  - 总仓位控制在预算范围内

第二层 (回撤控制 - 应对不利):
  - 预设最大可接受回撤
  - 动态仓位缩减: 仓位 = max(0, 1 - 当前回撤/最大可接受回撤)
  - 策略多元化

第三层 (灾难保护 - 防极端事件):
  - 限制最大杠杆
  - 定期压力测试
  - 保留应急现金
  - 硬性止损线 — 触及无条件清仓
```

**尾部风险量化**:
- **VaR**: "95%概率下最大亏损是多少" — 缺陷: 不知道剩下5%有多糟
- **CVaR / Expected Shortfall**: "最差5%情况的平均亏损" — 更好地捕捉尾部
- 真实市场收益率是**肥尾分布**，极端事件远比正态分布预测的频繁
- 2008年10月 S&P 单日跌9% 约 7-8 sigma，按正态分布"自宇宙诞生以来不应发生"

**LTCM 教训**: 两位诺贝尔奖得主，年化 40%，25-30 倍杠杆，1998 年四个月亏 46 亿。"杠杆不创造 Alpha，只放大风险"

**对我们的启示**:
- 我们的 P6 组合就是第二层"策略多元化"的实践
- MaxLoss=$30 是第三层"硬性止损"的体现
- R84 vol-sizing 是第一层仓位管理的改进方向

#### 系列（四）：多因子模型 — 华尔街印钱机器的核心秘密

**六大核心因子**:

| 因子 | 年化超额 | 表现好时 | 表现差时 |
|------|---------|---------|---------|
| 市场(Market) | ~7-8% | 长期 | 熊市 |
| 价值(Value) | ~3-5% | 经济复苏、利率上升 | 流动性泛滥、科技泡沫 |
| 规模(Size) | 近20年微弱 | 与质量/价值组合时 | 单独使用时 |
| 动量(Momentum) | ~6-8% | 趋势明确、波动适中 | 极端反转、危机结束 |
| 质量(Quality) | 稳定 | 经济下行、避险 | 风险偏好大开 |
| 低波动(Low Vol) | 反直觉正 | 市场下跌、恐慌 | 市场暴涨、情绪狂热 |

**因子动物园**: 学术报告 400+ 因子，Harvey et al. 2016 年指出绝大多数是数据挖掘产物。实践中只用 5-10 个核心因子。

**因子择时**:
> "因子择时极其困难，大多数尝试反而降低收益。保持恒定配置权重，定期再平衡。这听起来很无聊，但无聊恰恰是投资中最大的优势。" — AQR Asness

**因子组合力量**:
```
纯价值:           年化~10%, MaxDD~45%, Sharpe~0.4
纯动量:           年化~12%, MaxDD~50%, Sharpe~0.5
价值+动量+质量等权: 年化~11%, MaxDD~25%, Sharpe~0.8
→ 收益没降多少，风险大幅下降
```

**对我们的启示**:
- 我们的 P6 六策略组合正是多因子思想在 CTA 领域的实践
- 不要做策略择时/切换，保持恒定权重 + 定期再平衡
- R86 Risk Parity 是比等权更科学的"因子组合"方法

#### 系列（五）：回测年化 100%，实盘亏成狗 — 五大认知陷阱

| 陷阱 | 描述 | 我们的防御 |
|------|------|----------|
| **过拟合** | 模型记忆噪声而非学习规律。参数越多、数据越少越严重 | ①奥卡姆剃刀(参数最少化) ②严格样本外测试(3段分割) ③跨市场验证 ④t>3.0 门槛 |
| **幸存者偏差** | 只看到活下来的样本，不看死掉的 | ①数据包含退市标的 ②不只看"展示"的策略 ③R51 SuperTrend 否决就是反面验证 |
| **前视偏差** | 回测中使用了当时不可能获得的信息 | **已修复** — 2026-04-09 引擎修复: closed_only=True + 下根bar Open 执行 |
| **选择偏差/数据窥探** | 反复测试直到找到"有效"的。"超级碗指标"1967-1997准确率90%但毫无因果 | ①每个策略需有经济学逻辑 ②PBO 量化过拟合概率 |
| **黑天鹅** | "未知的未知"，模型之外的风险 | ①VaR/CVaR ②压力测试 ③保留现金 ④硬性止损线 |

**过拟合识别的四个信号**:
1. 参数异常敏感: 20日均线效果好，19日/21日大幅恶化 → 几乎肯定过拟合
2. 收益曲线过于完美: 几乎没有回撤 → 要么圣杯(概率≈0)要么过拟合
3. 策略逻辑缺乏经济学解释: "每周三下午2:30买入" → 无因果关系
4. 样本内外表现差异巨大: IS年化50% vs OOS年化5%

**我们的防御清单对照**:
```
策略开发阶段:
  ✅ 策略有清晰经济学逻辑 (KC突破+趋势)
  ✅ 参数数量控制在最小 (SL/TP/MH/Trail 共~6个)
  ✅ 严格样本外测试 (K-Fold 6/6)
  ✅ 参数鲁棒性 (±20% cliff test 无悬崖)
  ✅ t统计量检验 (DSR Stage 1)

回测阶段:
  ✅ 前视偏差已修复 (closed_only + next bar open)
  ✅ 加入真实交易成本 (spread $0.30-$1.00 压力测试)
  ✅ 回测期涵盖不同市场环境 (2015-2026, 牛熊震荡)

实盘阶段:
  ✅ 预设策略失效标准 (MaxDD > 2× 回测值)
  ✅ 独立风控层 (MaxLoss=$30)
  ⬜ 定期检查策略拥挤度 (未系统化)
```

#### 系列（六）：做量化为啥这么难？散户量化会被收割么？

**散户生存之道 — 与我们项目高度一致**:

#### 系列（七）：为什么所有公开的量化策略，都赚不了钱？

**策略公开悖论**: "策略一旦被公开，其价值即趋向于零"

**完整交易系统的五大模块**:
```
1. 信号生成 — 公开策略通常只包含这一项
2. 仓位管理 — 每笔交易的资金投入比例
3. 风险控制 — 止损规则、最大回撤限制、相关性控制
4. 执行逻辑 — 下单方式、滑点控制、部分成交处理
5. 市场适应 — 识别市场环境变化并动态调整
```

**我们的系统覆盖度**:
```

```

**核心引用**: "量化交易的核心不是策略本身，而是持续发现、开发、验证和迭代策略的能力 — 鱼竿而非鱼。"

### 2.2 TradingView 236 策略回测验证

**实验**: 对 236 个公开 TradingView 策略用 HyperLiquid 真实费率逐笔回测

**结果**:
```
236 个策略
 → 63 个通过严格复制验证 (26.7%)
   → 36 个盈利 (15.3%)
     → 21 个年化收益 >10% (8.9%)
```

**关键发现**:
- **交易频率是头号杀手**: 所有从盈利转为亏损的策略，年交易次数均超过 200 次
- **低频策略保住全部优势**: 最佳案例 90 天仅 16 笔交易
- 这完美印证了我们的低频策略方向

**对我们的启示**: 我们的 P6 组合年交易频率适中（L8_MAX 最高频约 1700 笔/年，但 D1KC 仅 ~47 笔/年），整体处于合理范围。

### 2.3 Agentic AI 驱动的自主量化因子挖掘

**来源**: [超越提示工程：Agentic AI 驱动的自主量化因子挖掘](https://zhuanlan.zhihu.com/p/2024942274769888809)

## 五、待深入研究方向

基于以上文献学习，以下方向值得后续深入：

1. **市场适应/策略退出机制**: Clever Liu 提到的完整系统第 5 模块。我们目前缺乏系统化的"策略失效检测"和"动态调整"机制。可以参考 regime detection 方法。

2. **CVaR 替代 VaR**: 在风控评估中引入 Conditional VaR，更好地量化尾部风险。

3. **策略拥挤度监控**: 定期评估我们的策略是否面临拥挤风险（黄金市场 KC 突破策略的使用者增长情况）。

4. **半凯利动态仓位**: 将凯利公式与 vol-sizing 结合，根据策略近期表现动态调整仓位比例。

5. **因子分解归因**: 用多因子框架分析 P6 组合收益的来源 — 多少是 Market β，多少是 Momentum α，多少是 Volatility α。
